4.1 INTRODUCTION TO BIG DATA
==================================================================

What is Big Data?

. Big Data refers to extremely large datasets that are difficult
  to store, process, and analyze using traditional systems.

        Examples:
        
                . Daraz customer orders
                
                . Amazon sales transactions
                
                . eSewa and Khalti payment records
                
                . Facebook posts and comments
                
                . Walmart retail transaction records




          Traditional Data Example:
          
                    . 100 customers
                    
                    . 500 sales transactions
          
          
          
          Big Data Example:
          
                  . 10 million customers
                  
                  . 500 million transactions

------------------------------------------------------------------
The 5 V's of Big Data
------------------------------------------------------------------

1. Volume

   . Large amount of data

   Example:

   . Amazon stores billions of transaction records.


2. Velocity

   . Speed at which data is generated

   Example:

   . Facebook receives thousands of posts every second.


3. Variety

   . Different types of data

   Examples:

   . Text
   . Images
   . Videos
   . Audio
   . Transaction Records


4. Veracity

   . Quality and reliability of data

   Example:

   . Duplicate customer records


5. Value

   . Business benefit obtained from data

   Example:

   . Identifying best-selling products




==================================================================
4.2 BIG DATA ANALYTICS TOOLS
==================================================================

A. Hadoop

      Purpose:
      
              . Store and manage large datasets across multiple computers.
      
      Features:
              
              . Distributed Storage
              
              . Fault Tolerance
              
              . Open Source
              
      Example:
      
              . Walmart stores petabytes of sales data using Hadoop.
      

------------------------------------------------------------------

B. Apache Spark

      Purpose:
      
              . Process Big Data very quickly.
      
      Features:
      
              . Faster than Hadoop MapReduce
              
              . Supports Machine Learning
              
              . Supports Real-Time Analytics
      
      Example:
      
              . Netflix analyzes millions of movie views using Spark.


------------------------------------------------------------------

C. Cloud Analytics Platforms

        Examples:
                
                . AWS (Amazon Web Services)
                
                . Google Cloud Platform (GCP)
                
                . Microsoft Azure
        
        Purpose:
        
                . Store and analyze data without buying servers.
        
        Example:
        
                . A company stores sales data in AWS S3
                  and analyzes it using cloud services.

n = 2500

regions = ["East", "West", "North", "South", "Central"]

segments = ["Consumer", "Corporate", "Home Office"]

categories = {
    "Technology": ["Laptop", "Printer", "Mouse", "Keyboard", "Monitor", "Router"],
    "Furniture": ["Chair", "Table", "Bookshelf", "Office Desk", "Sofa"],
    "Office Supplies": ["Paper", "Pen", "Notebook", "File", "Stapler", "Marker"]
}

customer_names = [
    "Ram", "Sita", "Hari", "Gita", "Shyam", "Nisha",
    "Bikash", "Anita", "Ramesh", "Sunita", "Kiran", "Puja"
]

rows = []

for i in range(1, n + 1):

    category = np.random.choice(
        list(categories.keys()),
        p=[0.38, 0.27, 0.35]
    )

    product = np.random.choice(categories[category])

    if category == "Technology":
        base_price = np.random.randint(1000, 90000)
        profit_rate = np.random.uniform(0.10, 0.28)

    elif category == "Furniture":
        base_price = np.random.randint(4000, 50000)
        profit_rate = np.random.uniform(-0.08, 0.18)

    else:
        base_price = np.random.randint(100, 8000)
        profit_rate = np.random.uniform(0.08, 0.30)

    quantity = np.random.randint(1, 6)

    discount = np.random.choice(
        [0, 0.05, 0.10, 0.15, 0.20, 0.30],
        p=[0.30, 0.20, 0.20, 0.15, 0.10, 0.05]
    )

    sales = round(base_price * quantity * (1 - discount), 2)

    profit = round(sales * profit_rate, 2)

    customer_id = f"C{np.random.randint(1, 401):04d}"

    customer_name = np.random.choice(customer_names)

    region = np.random.choice(regions)

    segment = np.random.choice(
        segments,
        p=[0.55, 0.30, 0.15]
    )

    order_date = pd.Timestamp("2025-01-01") + pd.to_timedelta(
        np.random.randint(0, 365),
        unit="D"
    )

    rows.append([
        f"O{i:05d}",
        order_date,
        customer_id,
        customer_name,
        segment,
        region,
        category,
        product,
        quantity,
        discount,
        sales,
        profit
    ])

df = pd.DataFrame(rows, columns=[
    "Order_ID",
    "Order_Date",
    "Customer_ID",
    "Customer_Name",
    "Segment",
    "Region",
    "Category",
    "Product",
    "Quantity",
    "Discount",
    "Sales",
    "Profit"
])

df.head()

]

    unique_products = sorted(set(products))

        pair_counter[pair] += 1

==================================================================
HADOOP VS SPARK
==================================================================

Hadoop:

. Stores very large data.

. Uses HDFS:
      Hadoop Distributed File System

. Data is divided into many blocks.

. Blocks are stored across many computers.


Apache Spark:

. Processes and analyzes large data quickly.

. Faster than Hadoop MapReduce.

. Used for:
      . Data analytics
      . Machine learning
      . Customer segmentation
      . Sales trend analysis
      . Recommendation system


Simple Difference:

. Hadoop = Big Data Storage

. Spark = Big Data Processing

import pandas as pd
import random

regions = ["East", "West", "North", "South", "Central"]
categories = ["Technology", "Furniture", "Office Supplies"]
products = ["Laptop", "Mouse", "Printer", "Chair", "Table", "Paper", "Pen"]

rows = []

for i in range(1, 100001):
    category = random.choice(categories)
    product = random.choice(products)
    region = random.choice(regions)
    sales = random.randint(500, 100000)
    profit = round(sales * random.uniform(-0.05, 0.25), 2)

    rows.append([i, region, category, product, sales, profit])

df = pd.DataFrame(rows, columns=[
    "Order_ID",
    "Region",
    "Category",
    "Product",
    "Sales",
    "Profit"
])

df.to_csv("big_retail_sales.csv", index=False)

print("100000 rows dataset created successfully")

from pyspark.sql import SparkSession
from pyspark.sql.functions import sum, avg, count

spark = SparkSession.builder \
    .appName("Big Data Retail Analytics") \
    .getOrCreate()

df = spark.read.csv(
    "big_retail_sales.csv",
    header=True,
    inferSchema=True
)

print("Dataset Schema")
df.printSchema()

print("Sample Data")
df.show(5)

print("Total Records")
print(df.count())

print("Sales by Category")
df.groupBy("Category") \
  .agg(
      sum("Sales").alias("Total_Sales"),
      sum("Profit").alias("Total_Profit"),
      count("Order_ID").alias("Total_Orders")
  ) \
  .show()

print("Sales by Region")
df.groupBy("Region") \
  .agg(
      sum("Sales").alias("Total_Sales"),
      sum("Profit").alias("Total_Profit")
  ) \
  .show()

print("Top Products by Sales")
df.groupBy("Product") \
  .agg(
      sum("Sales").alias("Total_Sales"),
      avg("Profit").alias("Average_Profit")
  ) \
  .orderBy("Total_Sales", ascending=False) \
  .show()

spark.stop()
